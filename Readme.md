## How to add exercises

- Add a new file in the appropriate folder (e.g., `Arrays/`, `BST/`, `Graph/`).
- Name files clearly (e.g., `problemName.cpp`) and keep small, focused examples.
- Commit with a short descriptive message.

## Notes

- This repo is for practice and learning; many files are standalone and intended to be compiled individually.
- The repository ignores user/editor files (see [.gitignore](.gitignore)).
